# Entropy Website

This project is a clean, educational website designed to introduce entropy across multiple levels of understanding. It focuses on clarity, accessibility, and strong visual structure without requiring prior knowledge of physics.

## Project Summary
This website explains the concept of entropy using simple language, diagrams, and brief examples. It is intended as both an educational resource and a demonstration of front-end development skills.

## Repository Link
[Entropy Website Repository](https://github.com/msamon00/entropy-website)

## Features
- Modern, responsive layout for all devices  
- Clear explanations for beginners through advanced readers  
- CSS-based visual design  
- Grid-based formatting for content organization  
- Optional embedded video demonstrations

## Purpose
The objective of this project is to help learners understand entropy in simple terms and to showcase strong web-design fundamentals without relying heavily on JavaScript.
